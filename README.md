# Serverless-REST-API-with-Cognito-Auth-DynamoDB-WAF
A production-ready serverless Management System built on AWS using API Gateway, Lambda, Amazon Cognito, DynamoDB, CloudFront, AWS WAF, and Amazon S3. The project demonstrates secure authentication, scalable REST APIs, and cloud-native architecture following AWS Well-Architected best practices.
